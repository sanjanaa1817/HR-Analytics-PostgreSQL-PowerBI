# HR Analytics Dashboard

## Project Overview

This project analyzes employee attrition and workforce trends using PostgreSQL and Power BI.

The objective was to identify factors affecting employee turnover, salary distribution, job satisfaction, and workforce demographics.

## Tools Used

- PostgreSQL
- Power BI
- SQL
- DAX

## Dataset

IBM HR Analytics Employee Attrition Dataset

Total Records: 1470 Employees

## SQL Analysis

Performed business-focused analysis using:

- Aggregations (COUNT, AVG, SUM)
- CASE Statements
- CTEs
- Window Functions (RANK)
- Attrition Analysis
- Salary Analysis
- Employee Segmentation

Key analyses included:

- Attrition Rate
- Attrition by Department
- Attrition by Overtime
- Salary Ranking
- Job Satisfaction Analysis
- Employee Promotion Analysis
- Department Salary Comparison

## Dashboard Features

### KPI Cards

- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Age

### Visualizations

- Attrition Rate by Department
- Attrition by Overtime
- Average Salary by Job Role
- Employee Distribution by Age Group
- Employee Attrition by Gender
- Job Satisfaction Analysis

### Interactive Filters

- Department
- Gender
- Overtime

## Key Insights

- Sales department recorded the highest attrition rate.
- Employees working overtime showed higher attrition.
- Managers and Research Directors received the highest average salaries.
- Most employees belonged to the 30–40 age group.
- Higher job satisfaction levels were associated with lower attrition.
